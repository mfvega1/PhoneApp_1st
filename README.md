## How to Use

1. Open `index.html` in a web browser
2. Use the **Dialer** tab to enter phone numbers manually
3. Click **Contacts** to quickly call specific departments
4. View your call history in the **History** tab
5. Press the green call button to initiate a call

## Departments & Contact Numbers

- **Sales Department** (555-0101) - For general inquiries and purchases
- **Delivery & Orders** (555-0102) - For delivery scheduling and tracking
- **Custom Arrangements** (555-0103) - For personalized flower arrangements
- **Customer Service** (555-0104) - For support and assistance
- **Wedding Flowers** (555-0105) - For wedding consultations
- **Event Planning** (555-0106) - For event flower arrangements
