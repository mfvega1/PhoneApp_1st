# PR Floral Marketing - Web Application

An artistic, flower-focused web application designed for the **PR Floral Marketing** brand. This mobile-optimized web showcase presents the brand's floral design and marketing services in an elegant, modern interface.

## Features

### Brand Identity
- Custom PR Floral Marketing logo with colorful spiral flowers
- Brand color palette featuring vibrant reds, pinks, purples, yellows, and greens
- Artistic floral motifs throughout the design
- Portrait/mobile-optimized layout (428px max width)

### Application Sections

1. **Home/Landing Section**
   - Prominent logo display with floating animation
   - Welcome message and brand tagline
   - Call-to-action buttons
   - Animated floral pattern backgrounds

2. **Gallery/Portfolio Section**
   - Showcase of floral arrangements
   - Filterable categories: All, Weddings, Events, Corporate
   - Smooth animations and transitions
   - Hover effects on gallery items

3. **Services Section**
   - Three main service offerings:
     - Custom Arrangements
     - Event Planning
     - Corporate Branding
   - Detailed descriptions with feature lists
   - Interactive service cards with hover effects

4. **Contact/About Section**
   - Brand story and business information
   - Contact form with validation
   - Business details (email, phone, hours, location)
   - Form submission with success notifications

### Technical Features

- **Pure HTML5, CSS3, and JavaScript** - No external dependencies
- **Responsive Design** - Optimized for portrait/mobile orientation
- **Smooth Animations** - Organic, flowing transitions inspired by floral movement
- **Interactive Elements** - Floral-inspired hover effects
- **Single Page Application** - Section-based navigation with smooth transitions
- **Form Validation** - Client-side form validation with user feedback
- **Accessibility** - Semantic HTML and proper ARIA attributes

## How to Use

1. **Local Development:**
   - Open `index.html` in a web browser
   - No build process or server required

2. **Deployment:**
   - Upload all files to any web server
   - Ensure proper MIME types are set for SVG files
   - Works with static hosting services (GitHub Pages, Netlify, Vercel, etc.)

3. **Customization:**
   - Brand colors are defined as CSS variables in `styles.css`
   - Update logo files in `assets/images/` directory
   - Modify business information in the Contact section of `index.html`
